Title: A / AN — God is Good, God is the Creator

# GRAMMAR
God is good. God is the creator. Look at the world! We talk about God’s creation in English. We use **a** and **an** to talk about one thing.

## 1. What is A / AN?
#IMG:imagem1.png
We use **a** or **an** before one thing.
We use **a** before consonant sounds.
We use **an** before vowel sounds (a, e, i, o, u).

TableStart
Word | Sentence
rabbit | This is a rabbit.
tree | This is a tree.
elephant | This is an elephant.
apple | This is an apple.
TableEnd

## 2. A with Consonants
#IMG:imagem2.png
Use **a** before words that start with consonant sounds.

TableStart
Word | Sentence
sun | This is a sun.
moon | This is a moon.
tree | This is a tree.
person | This is a person.
TableEnd

## 3. AN with Vowels
#IMG:imagem3.png
Use **an** before words that start with vowel sounds (a, e, i, o, u).

TableStart
Word | Sentence
elephant | This is an elephant.
apple | This is an apple.
orange | This is an orange.
egg | This is an egg.
TableEnd
