# GRAMMAR
Look! God made the world. God is good. We can look at things and ask: **What is this?** We can answer with **a** or **an**.

## 1. What is this?
#IMG:imagem1.png
We ask about one thing:
**What is this?**

We answer:
**This is a tree.**  
**This is an apple.**

TableStart
Question | Answer
What is this? | This is a tree.
What is this? | This is an apple.
What is this? | This is a rabbit.
TableEnd

## 2. A
#IMG:imagem2.png
We use **a** with many words.

Say:
**a tree**  
**a rabbit**  
**a person**  

TableStart
Word | Sentence
tree | This is a tree.
rabbit | This is a rabbit.
person | This is a person.
TableEnd

## 3. AN
#IMG:imagem3.png
We use **an** with words that start with A, E, I, O, U.

Say:
**an apple**  
**an elephant**  

TableStart
Word | Sentence
apple | This is an apple.
elephant | This is an elephant.
TableEnd

## 4. God made all things
#IMG:imagem4.png
Look at the world! God made everything.

We can say:
**This is a rabbit.**  
**This is an elephant.**  
God is the creator. God is good.

TableStart
Thing | Sentence
rabbit | This is a rabbit.
elephant | This is an elephant.
tree | This is a tree.
TableEnd

## 5. What is in your bag?
#IMG:imagem5.png
We can ask:
**What is in your bag?**

We answer:
**A book.**  
**An apple.**  

TableStart
Question | Answer
